# artw
Converts a Markdown subset to HTML

Implements:
- Headings
- Paragraphs
- Bold
- Italic
- Links
- Images
- Lists (ordered and unordered)
- Breaks
- Code sections

Note this is a hobby project, so this is *not* guaranteed to work with case like:
```md
- Item 1
1. Item 2
- Item 2
...
```

Or:
```md
- Hello
- *world*
```
